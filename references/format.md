## Category Guidance

- **🔥 重磅**：当天最重要、影响面最大的事件。包括重大模型发布、平台政策、公司战略、监管动作。
- **🚀 模型 & 技术突破**：模型、论文、基准、训练推理技术、开源项目、硬件软件协同。
- **💡 深度洞见**：行业人物观点、研究报告、战略判断、值得展开的争议。
- **💰 融资 & 商业**：融资、并购、收入、IPO、商业化、企业客户和价格策略。
- **🛠️ 工具 & 产品**：AI 应用、开发工具、插件、工作流产品、功能更新。
- **🔍 行业 & 治理**：政策监管、版权、安全、组织调整、人才流动、产业生态变化。

## Writing Standards

- Each item must include at least one full URL.
- Use direct URLs instead of Markdown hyperlinks so the report remains easy to copy across platforms.
- Prefer 18-25 selected items. Use 3-5 items for major sections and fewer for quiet categories.
- Do not include a personal QR code or private account promotion in the public version.
- If the user provides a brand footer or public subscription call-to-action, append it only when requested.

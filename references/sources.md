# Source And Verification Guide

## Search Plan

Use a combination of broad search, source-specific search, and direct source fetching.

Recommended queries:

- `AI news today YYYY-MM-DD`
- `artificial intelligence funding YYYY-MM-DD`
- `OpenAI Anthropic Google DeepMind Meta AI NVIDIA latest`
- `AI 模型 发布 YYYY年M月D日`
- `AI 融资 并购 监管 YYYY年M月D日`
- `site:openai.com blog AI`
- `site:anthropic.com news Claude`
- `site:deepmind.google blog`
- `site:nvidia.com AI announcement`
- `site:github.com AI release`
- `site:arxiv.org artificial intelligence`

For public figures, search both direct and media sources:

- Sam Altman / OpenAI
- Dario Amodei / Anthropic
- Demis Hassabis / Google DeepMind
- Jensen Huang / NVIDIA
- Yann LeCun / Meta AI
- Andrej Karpathy
- Ilya Sutskever
- Elon Musk / xAI

Only include a public figure item when the statement or event is current and traceable.

## Verification Rules

- **Dates**: Confirm the article date, announcement date, or repository release date.
- **Originality**: Prefer original announcements over rewritten summaries.
- **Numbers**: Funding amounts, valuations, benchmark scores, user counts, revenue, and prices require a trustworthy source.
- **Quotes**: Use direct posts, interviews, transcripts, or credible reports. Avoid quoting search snippets.
- **Benchmarks**: Include enough context to avoid overstating performance: task, comparison set, model version, and source.
- **Legal and safety claims**: Use official filings, regulator statements, company statements, or established outlets.

## Deduplication

Treat these as duplicates:

- Same model release covered by multiple outlets.
- Same funding round reported by different media.
- Same interview quoted by several newsletters.
- Same product update reposted by regional media.

Keep the source that is closest to the original event, then add a second source only when it improves confidence or context.

## Exclusion Rules

Exclude items that are:

- Older news repackaged as today's update.
- Pure rumors with no reliable source.
- Thin listicles without original reporting.
- SEO pages that provide no source trail.
- Product promotions with no meaningful news value.
